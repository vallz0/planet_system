## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/planet-simulation.git
   ```
2. Navigate to the project directory:
   ```sh
   cd planet-simulation
   ```
3. Install dependencies:
   ```sh
   pip install pygame
   ```

Run the simulation with:
```sh
python main.py
```
